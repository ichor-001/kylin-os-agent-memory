## Phase 1：领域调研（5月 - 7月）

- [ ] 精读近2年ACL/EMNLP/NeurIPS上的Peer Review相关论文
  - 如："Can LLMs Generate Structured Peer Reviews?" (ACL 2024)
  - "Reinforcement Learning for Reviewer Assignment"
  - "Multi-Agent Debate for Meta-Review Generation"
- [ ] 梳理当前研究的gap和open problems
- [ ] 确定自己的切入点（见下方"方向建议"）
- [ ] 产出：调研笔记 + 论文选题报告

### 方向建议（选一个深挖）

```
选项A：多Agent辩论式评审系统
  - 多个LLM Agent模拟不同审稿人，通过辩论达成共识
  - 可结合挑战杯的"记忆"模块，让Agent记忆审过的论文
  - 落地场景：期刊/会议的辅助审稿

选项B：Reviewer Confidence Calibration
  - LLM给出的审稿意见置信度校准
  - 解决LLM"过度自信"或"胡乱打分"的问题
  - 创新点：不确定性量化 + 校准算法

选项C：Meta-Review Generation with Memory
  - 基于多个单轮审稿意见生成综合的meta-review
  - 结合你在挑战杯做的记忆优化技术
  - 创新点：带记忆的多轮交互式meta-review

选项D：Persona-Driven Peer Review
  - 让LLM扮演不同学术背景的审稿人
  - 研究persona对审稿质量的影响
  - 创新点：可控的审稿persona建模
```
