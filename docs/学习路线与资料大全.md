## 二、第1阶段：基础扫盲（~2周）

### 2.1 AI Agent 是什么？（2-3天）

**必读论文：**
- Stanford **Generative Agents**（UIST 2023 Best Paper, 引用2000+）
  - arXiv: 2304.03442
  - Agent记忆系统的鼻祖，必读第一篇
- **MemGPT**（NeurIPS 2024 Oral, 引用600+）
  - arXiv: 2310.08560
  - OS风格虚拟内存管理LLM上下文

**推荐课程：**
- ⭐ **DeepLearning.AI - AI Agentic Design Patterns**（Andrew Ng）
  - URL: https://www.deeplearning.ai/short-courses/ai-agentic-design-patterns/
  - 时长：~2小时，免费
  - 覆盖：Reflection、Tool Use、Planning、Multi-agent 四种模式
  - 适合快速建立Agent的直觉
- ⭐ **DeepLearning.AI - Multi AI Agent Systems with CrewAI**
  - URL: https://www.deeplearning.ai/short-courses/multi-ai-agent-systems-with-crewai/
  - 时长：~2小时
  - 多个Agent协作的实际案例
- **Stanford CS224N Guest Talk: Generative Agents**
  - YouTube搜索: "Stanford CS224N Generative Agents"
  - 论文作者亲自讲解

**推荐博客/网站：**
- Lilian Weng (OpenAI) — "LLM Powered Autonomous Agents"
  - https://lilianweng.github.io/posts/2023-06-23-agent/
  - Agent技术全景扫描，每看一遍都有新收获
- Simon Willison's Blog — 实用Agent开发经验
  - https://simonwillison.net/tags/ai-agents/

### 2.2 记忆系统基础（2天）

**必读论文：**
- **Memory3**（arXiv 2025, 2407.01178）— 清华/百川
  - 三层架构：推理参数 + 显式知识记忆 + 短期工作记忆
- **Agent Memory Systems: A Survey**（搜索最新综述）
  - 关键词: "survey agent memory systems 2025"

**推荐课程：**
- **Hugging Face NLP Course** 第5章 "Transformers"
  - https://huggingface.co/learn/nlp-course
  - 重点读：Tokenization、Embedding基础
  - 免费，中文翻译版也有
- **Cohere's LLM University — Embeddings模块**
  - https://docs.cohere.com/docs/embeddings
  - 最清晰的embedding入门，带上手代码

**动手：**
- 你已经装好的TencentDB记忆系统是最好的教学环境
- 用 `curl` 调它的API完整走一遍流程
- 读它的README了解L0→L1→L2→L3 pipeline设计

### 2.3 Python + 数据处理基础（并行）

**推荐资源：**
- 《Python Data Science Handbook》前4章
  - 免费在线: https://jakevdp.github.io/PythonDataScienceHandbook/
- SQLite官方教程: https://sqlite.org/tutorial.html
- Python官方 `dataclasses` 文档: https://docs.python.org/3/library/dataclasses.html

## 三、第2阶段：核心技术逐个击破（~3周）

### 3.1 多源数据整合（要求1）—— 3天

**核心知识：** ETL pipeline设计、数据清洗、JSON Schema标准化、质量校验

**学习资源：**
- **Pydantic官方文档**: https://docs.pydantic.dev/
  - 数据建模+自动校验，赛题最实用的工具
- **Great Expectations**（可选）
  - https://greatexpectations.io/
  - 数据质量验证框架，赛题规模可能用不上，但思路可借鉴
- **Kaggle Learn — Data Cleaning**
  - https://www.kaggle.com/learn/data-cleaning
  - 1小时，免费
  - pandas数据清洗实战

**推荐课程：**
- **DeepLearning.AI - Building Systems with the ChatGPT API**
  - URL: https://www.deeplearning.ai/short-courses/building-systems-with-chatgpt/
  - 重点看：如何处理工具调用的返回结果
  - 时长：~1.5小时
- **Fast.ai - Practical Deep Learning** 第1课
  - https://course.fast.ai/
  - 讲如何从多种数据源构建pipeline

### 3.2 偏好记忆动态捕捉（要求2）—— 5天

**核心知识：** 偏好表示学习、动态更新、版本化管理、漂移检测、跨场景冲突

**必读论文（按顺序）：**
1. **MemoryBank**（AAAI 2024, 2305.10250）← 最直接
2. **Agent Workflow Memory / AWM**（ICLR 2025, 2409.07429）
3. **Reflexion**（NeurIPS 2023, 2303.11366）

**推荐课程：**
- ⭐ **DeepLearning.AI - Building and Evaluating Advanced RAG**
  - https://www.deeplearning.ai/short-courses/advanced-retrieval-for-ai/
  - 检索策略：查询重写、多路召回、重排序
  - 虽然讲的是RAG，但检索策略可直接迁移到偏好搜索
- **Stanford CS224U — Natural Language Understanding**
  - 完整课程: https://web.stanford.edu/class/cs224u/
  - 重点关注：Relation Extraction、Sentence Representation
  - YouTube有完整录像
- **Coursera - Recommender Systems Specialization**（University of Minnesota）
  - 可选，只看User Profiling部分
  - https://www.coursera.org/specializations/recommender-systems

**推荐博客：**
- "Concept Drift Detection" 入门
  - 搜索 "ADWIN"、"Page-Hinkley" 漂移检测算法
  - https://riverml.xyz/ — River ML 库的漂移检测教程
- "User Profiling with LLMs" 方法论
  - arXiv: 2407.01219 — "User Profiling in the Era of LLMs"

### 3.3 知识记忆结构化整合（要求3）—— 5天

**核心知识：** 轻量级知识图谱、冲突检测与消解、关联检索（不止找相似）

**必读论文：**
1. **HippoRAG**（NeurIPS 2024, 2405.14831）← 最核心
2. **AriGraph**（ACL 2024, 2407.04363）
3. **GraphRAG**（Microsoft, 2024）— 参考其思想但需轻量化

**推荐课程：**
- ⭐ **DeepLearning.AI - Knowledge Graphs for RAG**
  - https://www.deeplearning.ai/short-courses/knowledge-graphs-rag/
  - 时长：~1.5小时
  - 用Neo4j构建知识图谱增强RAG，最直接相关
  - 注意：赛题不用Neo4j（太重），但方法论通用
- **Stanford CS520 — Knowledge Graphs**
  - 完整课程: https://web.stanford.edu/class/cs520/
  - 斯坦福知识图谱正式课程
  - 重点看：Knowledge Graph Construction、本体设计
- **Coursera - Knowledge Graphs Specialization**（UC Davis）
  - https://www.coursera.org/specializations/knowledge-graphs
  - 更适合入门，可惜收费

**推荐网站/博客：**
- NetworkX官方教程: https://networkx.org/documentation/stable/tutorial.html
  - Python轻量图库，赛题端侧KG的首选
- 知乎搜索"知识图谱入门" — 中文教程很多
  - 推荐关注「刘焕勇」的知乎专栏，实战经验丰富
- "Conflict Resolution in Knowledge Graphs" 专题搜索

### 3.4 端侧适配（要求4）—— 3天

**核心知识：** 向量搜索原理、Embedding模型对比、索引优化、500ms硬约束

**推荐课程：**
- ⭐ **Cohere LLM University — Search模块**
  - https://docs.cohere.com/docs/text-search
  - 语义搜索从零到一
- **Pinecone Learning Center — Vector Database 101**
  - https://www.pinecone.io/learn/vector-database/
  - 向量数据库原理：HNSW、IVF、PQ量化
- **Stanford CS224N — Lecture 10: Question Answering**
  - 检索系统的评估方法论

**推荐网站：**
- **MTEB Leaderboard**（Hugging Face）
  - https://huggingface.co/spaces/mteb/leaderboard
  - 看小模型在小榜单上的表现，选适合端侧的embedding
- **sqlite-vec** 项目
  - https://github.com/asg017/sqlite-vec
  - 零依赖的SQLite向量搜索插件，端侧首选
- **Hugging Face Model Search** — 中文小模型
  - 搜索"text2vec-base-chinese"、"shibing624/text2vec-base-chinese"

**性能优化参考：**
- "Efficient Nearest Neighbor Search" 综述
- "ANN Benchmarks" (https://ann-benchmarks.com/)

### 3.5 安全与遗忘（要求5）—— 2天

**核心知识：** 敏感信息识别、NL指令→精确删除、精准遗忘策略

**学习资源：**
- **Microsoft Presidio**

  - PII检测框架，规则+ML双模式
  - 可作为敏感信息过滤的直接方案
- **Stanford CS224N — Lecture 6: Constituency Parsing**

**推荐博客：**
- "The Right to be Forgotten in ML Systems" 概念理解
- "Machine Unlearning" 综述（2024）

### 3.6 记忆流转（要求6）—— 1天

**核心知识：** 短期→中期→长期流转策略、时间衰减、重要性评分

**学习资源：**
- Generative Agents的memory stream框架
- MemGPT的分页管理
- TencentDB的pipeline机制就是参考

**推荐工具：**
- Redis（可选）— 适合做短期记忆的热存储
- SQLite — 首选，长期记忆用

### 3.7 量化评测（要求7）—— 2天

**核心知识：** IR评估指标、precision/recall/F1、MAP、NDCG

**推荐课程：**
- **Coursera - Advanced NLP**（by Yandex）
  - https://www.coursera.org/learn/advanced-nlp
  - 重点看Information Retrieval评估部分
- **Stanford CS276 — Information Retrieval**
  - 完整课程: https://web.stanford.edu/class/cs276/
  - IR的黄金标准课程

**推荐网站：**
- "Introduction to Information Retrieval" 在线版
  - https://nlp.stanford.edu/IR-book/
- Kaggle上搜索"information retrieval evaluation" notebook

### 4.1 论文获取渠道

| 渠道 | 网址 | 说明 |

| **Hugging Face Papers** | https://huggingface.co/papers | 每日论文推荐 |
| **AlphaXiv** | https://alphaxiv.org/ | LLM优化的论文摘要+讨论 |
| **OpenReview** | https://openreview.net/ | 含审稿意见，可看同行评价 |

  - https://www.zotero.org/
- **Connected Papers** — 论文关系图谱
  - https://www.connectedpapers.com/
- **Elicit** — AI辅助论文搜索+提炼
  - https://elicit.com/

### 4.3 精读优先级

| 优先级 | 论文 | arXiv | 关联要求 |
|--------|------|-------|----------|
| ⭐⭐⭐ | MemGPT | 2310.08560 | 记忆架构标杆 |
| ⭐⭐⭐ | HippoRAG | 2405.14831 | 要求3，KG+检索 |
| ⭐⭐⭐ | MemoryBank | 2305.10250 | 要求2，偏好记忆 |
| ⭐⭐ | AWM | 2409.07429 | 要求2 |
| ⭐⭐ | GraphRAG | 2404.16130 | 要求3，需轻量化 |
| ⭐⭐ | Generative Agents | 2304.03442 | 奠基性工作 |
| ⭐ | Memory3 | 2407.01178 | 架构参考 |
| ⭐ | AriGraph | 2407.04363 | 动态KG |
| ⭐ | Reflexion | 2303.11366 | 反思机制 |

### 4.4 精读模板

每篇论文花1-2天：

```
1. 读标题+摘要+结论（10分钟）→ 判断相关性
2. 读方法部分（1小时）→ 核心创新点是什么？
3. 看实验设置（30分钟）→ 评估标准、数据集
4. 画一张图总结（30分钟）→ 强迫自己理解
5. 写一句话：这篇论文和赛题要求X的关系？
```

## 最后提醒：3条最重要的原则

**1. 别花太多时间「学习」，尽快进入「动手」**

赛题要的是能跑的作品，不是考试分数。读到"够用"就停下来动手。TencentDB已经给了你80%的基础设施。

**2. 麒麟SDK是关键风险点，提前催**

要求(4)核心是"调用银河麒麟embedding SDK"。6月初没拿到SDK的话，必须准备替代方案（用开源的 text2vec-base-chinese 等小模型做fallback）。建议现在就让王雄老师帮忙联系赛事方。

**3. 技术创新性30分是特等奖的突破口**

常规方案只能拿基础分。要想特等/擂主，必须有创新点：
- 可能是"LLM驱动的偏好推理"（不等人说就主动推断变化）
- 可能是"增量知识图谱的轻量化冲突消解"
- 可能是"多级遗忘曲线索引"（时间+频率+重要性三维调度）

现在多读论文，就是在为这一步储备弹药。
