## 🧠 赛题核心

**命题**：在麒麟OS环境下，研究并实现高效的Agent记忆系统，优化Agent的长期记忆管理与高效应用。

**核心指标**：
- 召回率 ≥ 85%
- 响应时间 ≤ 500ms  
- 冲突正确率 ≥ 88%
- 偏好准确率 ≥ 85%

## 📊 技术路线

本方案基于 **符号化短期记忆 + 分层长期记忆 (L0-L3)** 的四层记忆金字塔架构：
- **L0**: 对话原文存储
- **L1**: 原子事实提取
- **L2**: 场景块聚类
- **L3**: 用户画像合成

结合时序知识图谱 + 遗忘曲线 + 轻量RL选择器，实现OS级Agent记忆管理。
